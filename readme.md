## 项目目录

```
src                                  源码目录
|-- api                              接口
|-- assets                           静态资源
|-- components                       公用组件
|-- utils                            工具类函数
|-- router                           路由，统一管理
|-- styles                           样式目录
|-- views                            视图目录
|-- main.js                          项目的核心文件
|-- App.vue                          项目入口文件
```
